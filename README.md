# PtiCalin/sql — Le référentiel SQL personnel

Ce répositoire centralise toutes mes connaissances, ressources, exemples, exercices et outils liés à SQL, dans une structure claire, évolutive et facile à naviguer.

---

## Objectif

Ce dépôt vise à :
- Documenter la théorie SQL (syntaxe, concepts, bonnes pratiques)
- Illustrer par des exemples concrets et annotés
- Proposer des exercices pratiques (avec solutions, puis interface interactive à terme)
- Recenser et commenter des ressources externes (articles, vidéos, docs)
- Lister et documenter des outils SQL utiles

---

## Structure

```
theory/      # Concepts et explications fondamentales
examples/    # Exemples de requêtes et cas d'usage
exercises/   # Exercices pratiques et solutions
resources/   # Liens, articles, vidéos, docs commentés
tools/       # Outils, scripts et utilitaires SQL
```

---

## Organisation & Évolutivité

- **Ajouter** chaque nouvelle notion, ressource ou outil dans le dossier approprié.
- **Commenter** vos ressources : pourquoi sont-elles utiles ? Points clés ?
- **Garder** une structure claire : titres explicites, navigation facile.
