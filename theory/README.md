# Théorie — Base de données relationnelles

Ce dossier contient les fondements conceptuels, organisés selon la progression naturelle de l'apprentissage. Chaque module construit sur le précédent.

## Objectifs généraux

À l'issue de ce parcours théorique, vous serez capable de :
- Expliquer le modèle relationnel et ses propriétés fondamentales
- Concevoir un schéma de BD à partir d'un énoncé en langage naturel
- Appliquer l'algèbre relationnelle pour raisonner sur des requêtes
- Écrire des requêtes SQL correctes et efficaces (DDL, DML, DQL)
- Normaliser un schéma jusqu'à la FNBC
- Utiliser les objets SQL avancés (vues, procédures, fonctions, curseurs)
